# Agent Guide — Mieru Series

## Mission

このディレクトリを読んだエージェントは、ミエルシリーズについて次を判断できる状態にする。

- 誰に売るか
- 何を売るか
- いくらで売るか
- どう診断するか
- どう提案するか
- 受注後30日で何を納品するか

## Core Positioning

ミエルシリーズは、安いスケジュール管理アプリではない。

現場会社の「見えない損失」を、事務所TVとスマホ入力で見える化する導入支援付きパッケージ。

## Main Phrase

現場の「どこいった？」をなくす。

## Product Family
