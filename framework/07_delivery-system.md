# 07 Delivery System

Delivery System は、受注後の進行、納品、継続提案を標準化するモジュールです。

## 目的

受注後の流れを整え、顧客満足、継続契約、次の商品への導線を作ります。

## 管理するもの

- ヒアリングシート
- 納品テンプレート
- 成果物フォーマット
- オンボーディング
- フォローアップ
- 次の商品への案内
- 事例化テンプレート

## 出力

- `products/{product_id}/delivery-template.md`
- `products/{product_id}/case-study-template.md`
- `products/{product_id}/next-offer.md`

## ルール

- 納品内容を毎回ゼロから作らない
- 納品後に次の商品へ自然につなげる
- 顧客の声を取得する
- 匿名事例化できる条件を事前に確認する
