# 08 Feedback Loop

Feedback Loop は、数値・反応・失注理由を次の改善に戻すモジュールです。

## 目的

発信、LP、商談、納品で得た反応を記録し、次のオファー改善に反映します。

## 見る指標

- 投稿数
- 反応数
- LP訪問数
- フォーム入力数
- 商談数
- 受注数
- 受注率
- 平均単価
- 失注理由
- 成約理由
- 納品満足度
- 継続率
- 紹介数

## 出力

- `products/{product_id}/metrics.md`
- `reports/weekly/{date}-{campaign_id}.md`
- `products/{product_id}/objections.md`

## ルール

- 数字と感想を分ける
- 反応がない場合も記録する
- 顧客の言葉をそのまま残す
- 改善タスクは5個以内に絞る
- 次の週に必ず反映する
