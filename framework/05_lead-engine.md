# 05 Lead Engine

Lead Engine は、見込み客を集める導線を設計するモジュールです。

## 目的

発信、無料資料、診断フォーム、DM、紹介を組み合わせて、商談につながるリードを作ります。

## 基本導線

```text
発信
  ↓
無料チェックリスト / 診断
  ↓
フォーム入力
  ↓
適格性確認
  ↓
有料商品
```

## チャネル

- X
- note
- YouTube / ショート動画
- 既存知人
- 紹介
- 直接DM
- GitHub / 開発発信
- セミナー

## 出力

- `products/{product_id}/lead-magnet.md`
- `products/{product_id}/diagnosis-form.md`
- `campaigns/{campaign_id}/dm-script.md`

## ルール

- すぐ売り込まず、問題を教育する
- 無料導線は顧客が実際に使えるものにする
- DM送信や個別連絡は人間承認を必須にする
- リードの質を見て、対象者を絞る
