# 09 Pitch Compressor

Pitch Compressor は、Offer Compiler で作ったオファーを、短く伝わる言葉に圧縮するモジュールです。

## 目的

ホルモジ式で作った強いオファーを、Xプロフィール、固定ポスト、LPファーストビュー、DM冒頭、商談冒頭で使える短い表現に変換します。

## 位置づけ

```text
Offer Compiler
  ↓
Pitch Compressor
  ↓
Asset Factory
```

## 作るもの

- 1行ピッチ
- 15秒ピッチ
- 30秒ピッチ
- 90秒ピッチ
- 3分ピッチ
- LPファーストビュー
- DM冒頭文
- 商談冒頭文

## 入力

- `products/{product_id}/product.yaml`
- `products/{product_id}/offer.md`

## 出力

- `products/{product_id}/pitch.md`

## 基本ルール

- 誰向けかを先に出す
- 何の問題を解くかを明確にする
- 技術名より損失回避・成果・時間短縮を優先する
- 長い説明ではなく、会話で言える言葉にする
- 誇大表現を避ける
- 実績がないものを実績として言わない

## ピッチの型

### 1行ピッチ

```text
{対象者}向けに、{痛み}を{短い時間/具体手段}で解決する{商品カテゴリ}です。
```

### 15秒ピッチ

```text
{対象者}が{よくある失敗}を避けるために、{提供内容}を{時間}で整理します。
```

### 30秒ピッチ

```text
{市場/対象者}では、{問題}が起きています。
このサービスでは、{対象案件/対象状況}について、{診断項目}を整理し、{次の行動}まで落とし込みます。
```

### 90秒ピッチ

1. 誰向けか
2. 何が問題か
3. 放置すると何が起きるか
4. 何を提供するか
5. どう進めるか
6. 次に何をすればよいか

### 3分ピッチ

1. 市場背景
2. 顧客の痛み
3. 既存の代替手段の限界
4. オファー
5. 納品物
6. 価格と制約
7. 次アクション

## 使用場所

| ピッチ | 使用場所 |
|---|---|
| 1行 | Xプロフィール、サービス一覧、DM見出し |
| 15秒 | 自己紹介、固定ポスト冒頭 |
| 30秒 | 商談冒頭、LPファーストビュー |
| 90秒 | note冒頭、紹介依頼 |
| 3分 | 商談導入、セミナー冒頭 |
